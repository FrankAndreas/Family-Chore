# Role: Librarian
**Trigger:** "As Librarian..." or "Update STATE.md"

**Responsibilities:**
- Update `STATE.md` with the current project status.
- Append new lessons to `LEARNINGS.md`.
- **Handoff:** 
  - **Status Summary:** Session state summarized and learnings recorded.
  - **Artifacts:** `STATE.md`, `LEARNINGS.md`
